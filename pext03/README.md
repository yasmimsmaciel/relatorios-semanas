# Compilador TeX para PDF

## 1. Introducao

Este projeto foi desenvolvido no contexto da disciplina **Programacao 2**, do curso de Engenharia de Controle e Automacao da Universidade de Pernambuco (UPE), sob orientacao do Prof. Dr. Ruben Carlo Benante.

O objetivo central consiste na implementacao de um sistema capaz de processar arquivos no formato **TeX** e gerar como saida documentos em **PDF**, explorando conceitos fundamentais da linguagem C e ferramentas associadas ao ecossistema LaTeX.

---

## 2. Objetivos

### 2.1 Objetivo Geral

Desenvolver um compilador simplificado de arquivos TeX para PDF utilizando a linguagem C.

### 2.2 Objetivos Especificos

* Aplicar conceitos de programacao em C, incluindo manipulacao de arquivos e estruturas de dados;
* Automatizar o processo de compilacao de documentos utilizando Makefile;
* Integrar ferramentas como LaTeX e BibTeX;
* Produzir material didatico complementar (relatorio e videos explicativos).

---

## 3. Tecnologias Utilizadas

* Linguagem C
* LaTeX
* BibTeX
* Makefile

## 5. Compilacao e Execucao

Para compilar o documento principal, execute:

```

```

O projeto contem:
