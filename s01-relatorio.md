Relatorio 01

1. Objetivo
Nesta primeira semana, foi definido para o projeto a escolha dos temas a serem desenvolvidos pelo grupo.

2. Atividades Desenvolvidas
Apos a definicao dos integrantes do grupo, foi realizada uma reuniao para a discussao de temas para a elaboracao dos videos, analisando as propostas dos integrantes do grupo bem como a viabilidade de cada uma delas para a atividade proposta.

3. Metodologia Utilizada
- Reunioes em grupo;
- Tomada de decisoes em consenso.

4. Resultados
Foram definidos o tema do projeto, alinhando junto aos objetivos do mesmo e os proximos passos para a execucao desta atividade.

5. Planejamento para a proxima semana
Depois de definido o tema do projeto, para a proxima semana se estabeleceu que seriam discutidos de que forma o grupo executaria todos os itens solicitados pelo professor, bem como a divisao das atividades entre os integrantes do grupo.
